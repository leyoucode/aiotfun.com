## 这是什么

中小快递公司没有顺丰 / 京东那样的运营系统，只能用 Excel 和微信群凑活。吉时达把"接单 → 派件 → 签收 → 对账"串成一条标准化流程，APP / Web / 司机端三个入口。

## 关键挑战

- **断网作业**：司机在山区 / 地下车库经常断网，App 必须能完整离线工作，回到信号区再同步。
- **多角色权限**：商户 / 派件员 / 调度 / 对账，每种角色看到的字段都不一样。
- **实时追踪**：用 LBS + 地图 SDK 做派件轨迹，但要平衡功耗和位置精度。
